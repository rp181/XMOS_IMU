void readGPS ( chanend  uartRX,
chanend  gps,
unsigned  baud_rate 
)

Starts the thread to read the GPS data, as well as handle data requests. Requests are always dealt with first, pausing information update. As a result, the rate of requests should be limited, to allow for update.

Parameters:
uartRXUART data from the UART thread
gpschannel to take and respond to requests
baud_ratebaud rate that the uart thread is running
